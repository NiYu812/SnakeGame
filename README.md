# 贪吃蛇游戏 - 使用C++和Qt开发

## 项目概述
这是一个使用C++和Qt框架开发的经典贪吃蛇游戏。项目采用面向对象设计，包含完整的游戏逻辑、UI界面和交互功能。

## 功能特性
- 🎮 经典贪吃蛇游戏玩法
- 🎨 美观的图形界面
- ⌨️ 键盘控制（方向键或WASD）
- 📊 实时得分显示
- ⏸️ 暂停/继续功能
- 🎯 游戏结束提示
- 🎲 随机食物生成
- 🧱 碰撞检测（墙壁和自身）

## 编译和运行

### 环境要求
- Qt 5.x 或 Qt 6.x 开发环境
- CMake 3.5 或更高版本
- C++编译器（MSVC, GCC, Clang等）

### 编译步骤

#### 方法1：使用Qt Creator（推荐）
1. 打开Qt Creator
2. 选择"打开项目"，选择CMakeLists.txt文件
3. 配置构建套件（Kit）
4. 点击"构建"按钮
5. 点击"运行"按钮启动游戏

#### 方法2：使用命令行
```bash
# 创建构建目录
mkdir build
cd build

# 配置CMake（确保Qt已在系统路径中）
cmake .. -DCMAKE_PREFIX_PATH="你的Qt安装路径"

# 编译项目
cmake --build . --config Release

# 运行游戏
./Release/SnakeGame.exe
```

### 键盘控制
- **方向键** 或 **WASD**：控制蛇的移动方向
- **空格键**：暂停/继续游戏
- **ESC**：退出游戏

## 游戏规则
1. 控制蛇吃红色的食物来获得分数
2. 每吃一个食物得10分，蛇身会变长
3. 不能撞到墙壁或自己的身体
4. 游戏结束后可以选择重新开始

## 代码说明

### 主要类结构

#### MainWindow类
- 主窗口界面
- 菜单和状态栏
- 游戏控制按钮
- 键盘事件处理

- 食物生成

#### GameScene类
- 游戏场景绘制
- 网格、蛇身和食物的渲染
- 图形界面更新
